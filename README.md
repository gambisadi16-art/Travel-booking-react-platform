# ✨ LuxeStay – Luxury Hotel Booking Platform

LuxeStay is a modern hotel booking platform built with **React**, designed to deliver a premium travel experience through elegant design, smooth animations, and intuitive functionality. Inspired by platforms like Airbnb and Booking.com, LuxeStay combines luxury aesthetics with user-friendly interactions to help travelers discover and explore exceptional accommodations.

The project focuses on creating a visually immersive experience while maintaining fast performance, responsiveness, and a clean component-based architecture.

---

## 🌟 Features

- 🏨 Premium hotel booking interface
- 🔍 Interactive hotel search experience
- 🌍 Featured destination showcases
- 🛏️ Curated luxury hotel listings
- 📱 Fully responsive design
- 🎬 Smooth animations with Framer Motion
- 🎨 Modern UI built with Tailwind CSS
- ⚡ Fast and efficient performance using Vite
- 🧩 Reusable React components
